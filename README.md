# Ferrari Strategy 🏎️

[![CI](https://github.com/imaO0O/ferrari-strategy/actions/workflows/ci.yml/badge.svg)](https://github.com/imaO0O/ferrari-strategy/actions/workflows/ci.yml)

> Живой фан-хаб Формулы-1 в цветах Ferrari: статистика сезона, браузерная
> телеметрия с реплеями гонок, энциклопедия Ф1 с 1950 года, игры и обучение —
> бесплатно, без регистрации и без рекламы.

**Автор:** [imaO0O](https://github.com/imaO0O)  ·  **Демо:** https://ferrari-strategy.vercel.app/

Ferrari Strategy решает две задачи сразу. Как **портфолио** — показывает работу
с живыми API, управление состоянием, визуализацию данных, 3D и доступность.
Как **продукт** — закрывает реальную нишу: нормального русскоязычного фан-сайта
Формулы-1 такого объёма сейчас нет. Всё работает **полностью на клиенте**: нет
бэкенда, серверов, ключей API и куки — личные данные (рекорды, прогнозы,
любимый пилот) хранятся только в браузере.

## Что умеет

**Статистика и архив**
- Живой зачёт сезона — пилоты, Кубок конструкторов, календарь с обратным отсчётом.
- **Машина времени** — любой сезон Ф1 с 1950 года: зачёты, результаты, дуэли, пересчёт по разным очковым системам.
- **Энциклопедия** — авто-профиль любого пилота, команды и трассы в истории Ф1: карьера по сезонам, победы, победители Гран-при.

**Телеметрия (данные OpenF1)**
- Браузерный реплей гонки: башня позиций, счётчик кругов, перемотка и скорость воспроизведения.
- Стратегия шин, «история гонки» (гэпы по кругам), сравнение темпа двух пилотов, пит-стопы, погода, карта трассы.
- **Live-центр** — второй экран уик-энда: расстановка, карта и радио Ferrari в реальном времени.

**Интерактив**
- Прогнозы на подиум — очки начисляются автоматически после гонки.
- Симулятор чемпионата: расставь результаты оставшихся этапов и смотри проекцию титула.
- **Дуэль** пилотов — в рамках сезона или карьер легенд разных эпох.
- Экспорт календаря сезона в `.ics` для Google/Apple Calendar.

**Контент и игры**
- Интерактивная история Скудерии с 1947 года, досье легенд и гараж культовых болидов.
- **Академия** — Формула-1 за 15 минут для новичков плюс глоссарий терминов.
- Четыре игры (реакция на старт, пит-стоп, викторина, «угадай трассу» по реальным GPS-контурам), достижения и PNG-карточки результатов.
- **Паспорт тифози** — весь личный прогресс в одном месте, с экспортом и импортом файлом.

## Как это устроено

- **Без бэкенда.** Одностраничное React-приложение целиком в браузере: ноль серверов, ключей и переменных окружения.
- **Приватность по умолчанию.** Нет куки, трекеров и рекламы; рекорды и прогнозы — только в `localStorage`.
- **Данные.** [Jolpica F1 API](https://jolpi.ca) (результаты и зачёты с 1950 года) и [OpenF1](https://openf1.org) (телеметрия, радио, погода). Фото — [Wikimedia Commons](https://commons.wikimedia.org) с атрибуцией, подгружаемой из API.
- **Качество кода.** Юнит-тесты (Vitest) и CI на GitHub Actions, глобальный `ErrorBoundary`, ленивые чанки страниц, доступность (клавиатурный фокус, skip-link, ARIA, `prefers-reduced-motion`), уникальные SEO-мета плюс `sitemap.xml` и `robots.txt`.
- **Дизайн-система.** Карбон и rosso corsa, шрифты Exo 2 и Orbitron, кастомный курсор, панельные переходы страниц и процедурный 3D без сторонних ассетов.

## Технологии

| Слой | Инструменты |
|---|---|
| Каркас | React 18 + Vite 6 |
| Стили | Tailwind CSS v4 (токены в `src/styles/index.css`) |
| Анимации | Framer Motion, Lenis (плавный скролл) |
| 3D | Three.js + React Three Fiber + drei |
| Данные | Jolpica F1 API, OpenF1 API — бесплатно, без ключей |
| Тесты / CI | Vitest, GitHub Actions |
| Шрифты | Exo 2 (кириллица) + Orbitron — Google Fonts |

```
src/
  pages/       страницы-маршруты: Скудерия, Дашборд, Гонки, Телеметрия,

public/        robots.txt, sitemap.xml, favicon
```
